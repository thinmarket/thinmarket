Как работает?

Вариант 1 (Классика)

Лонг:
  • Берём две последние закрытые недели.
  • Если закрытие недели 2 выше закрытия недели 1 — тикер попадает в список.
  • Жёлтым выделяются те, у кого закрытие недели 2 не выше недели 1, но текущая цена выше хая недели 2 (breakout по текущей цене).
  • Глубокий анализ: фильтрует только те тикеры, где текущая цена выше хая недели 2.
Шорт:
  • Берём две последние закрытые недели.
  • Если закрытие недели 2 ниже закрытия недели 1 — тикер попадает в список.
  • Жёлтым выделяются те, у кого закрытие недели 2 не ниже недели 1, но текущая цена ниже лоу недели 2 (breakdown по текущей цене).
  • Глубокий анализ: фильтрует только те тикеры, где текущая цена ниже лоу недели 2.

Вариант 3 (Экстремум за полгода)

Лонг:
  • Берём 26 последних недель (полгода).
  • Находим максимальный хай за этот период.
  • В список попадают тикеры, у которых закрытие последней недели выше этого хая.
  • Жёлтым выделяются те, у кого закрытие недели не выше хая полугодия, но текущая цена выше этого хая.
  • Глубокий анализ: фильтрует только те тикеры, где текущая цена выше максимального хая за полгода.
Шорт:
  • Берём 26 последних недель (полгода).
  • Находим минимальный лоу за этот период.
  • В список попадают тикеры, у которых закрытие последней недели ниже этого лоу.
  • Жёлтым выделяются те, у кого закрытие недели не ниже лоу полугодия, но текущая цена ниже этого лоу.
  • Глубокий анализ: фильтрует только те тикеры, где текущая цена ниже минимального лоу за полгода.

Общие пояснения

  • Жёлтый цвет — breakout/breakdown по текущей цене, если не выполнено основное условие по закрытию.
  • Кнопка "Глубокий анализ" фильтрует только те тикеры, где текущая цена уже пробила нужный уровень (high/low/экстремум).
  • В таблице можно фильтровать по типу инструмента (акция, фьючерс, валюта).
  • Для акций доступен переход на график TradingView.
